netop

htop-style TUI for network traffic. Sort by RX/TX per process/connection. Uses eBPF on Linux.

Install notes

  • Linux: cargo install netop (richiede kernel 5.x+ con BPF, headers)
  • macOS: support limitato (no eBPF); funziona via fallback ma feature ridotte
  • Alternativa cross-platform più matura: bandwhich o nethogs (Linux only)

Top commands

  • sudo netop — launch (root serve per eBPF capture)
  • netop --help — flag list
  • Inside TUI:
  • q quit
  • sort: tab between RX/TX/total
  • filter by PID/process name

Pitfalls

  • Root required (eBPF + raw sockets)
  • macOS: usa bandwhich come alternativa (brew install bandwhich) — più stabile per Mac
  • No brew formula; cargo-only install
  • Verifica versione recente in repo — progetto giovane
