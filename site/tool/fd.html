fd

Simpler find. Regex syntax, colored output, parallel.

Top commands

  • fd PATTERN — find files matching regex in cwd
  • fd -e py — by extension
  • fd -t d PATTERN — directories only
  • fd -H PATTERN — include hidden
  • fd -I PATTERN — include gitignored
  • fd PATTERN -x cmd {} — exec per match
  • fd -X cmd — batch exec
  • fd --changed-within 1d — recent files
  • fd -s PATTERN — case sensitive

Pitfalls

  • Default regex, not glob — use -g for glob
  • Respects .gitignore — use -I to disable
